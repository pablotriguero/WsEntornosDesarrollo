
# JavaDoc

El objetivo de javadoc es el de proporcionar informacion a otros desarrolladores de como tienen que utilizar nuestros clases y metodos. Esta documentacion no es para nosotros (que tambien) es principalmente para los programadores que usen nuestro código.

## Generar documentacion

Este es un ejemplo basico de como generar documentación,
Una vez documentado el codigo podemos generar documentación
en html de la siguiente manera
	Proyect -> Generate JavaDoc
	
A continuacion elegiremos el proyecto que queremos generar su java doc

Si nos da un error de tipo "javadoc command does not exist" debemos
de buscar en dicha pantalla la ruta del ejecutable para que nos genere la documentación
El ejecutable que podemos escoger por defecto seria "javadoc.exe" que se encuentra donde
tengas instalado nuesta jdk de java. Por ejemplo podría ser:

C:\Program Files\Java\jdk1.8.0_231\bin
